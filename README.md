# UniMate

An MSc Advanced Computer Science research prototype by Onkar Anantrao Yelchalwar.

UniMate is an Alexa-based conversational study assistant. The foundation release
supports a launch interaction, concise topic explanations, and short topic quizzes.

## Repository structure

- `backend/`: AWS Lambda source code.
- `tests/`: automated checks for small units of logic.
- `docs/`: setup, testing, and dissertation evidence.
- `diagrams/`: architecture and UML diagrams created during design.
- `assets/`: non-sensitive screenshots and other visual evidence.

The initial Alexa interaction model is in `docs/alexa-interaction-model.json`.
